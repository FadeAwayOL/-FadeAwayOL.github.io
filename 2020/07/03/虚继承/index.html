继承

多重继承

定义

一个派生类具有两个或两个以上的直接基类就是多重继承

  • 1 一个子类可以拥有多个父类；

  • 2 子类拥有所有父类的成员变量；

  • 3 子类继承所有父类的成员函数；

  • 4 子类对象可以当作任意父类对象使用；

    格式

多重继承派生类的格式

class 派生类名：继承方式1 基类名1，继承方式2 基类名，….

{ 派生类类体 }；

多重继承派生类的构造函数格式

派生类名（总参数表）：基类名1（参数表1），基类名2（参数表2）,…

子对象名（参数表n+1），…//如果有子对象的话

{ 派生类函数体 }
